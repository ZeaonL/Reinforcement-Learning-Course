# this is zyliu's Reinforcement Learning Course procedure
## 开发环境
1. 安装anaconda
2. 在conda创建一个新的环境rl
3. 安装Git
4. 安装模块:
    * pytorch
    * swig（不知道有什么用）
    * jupyter
    * git clone https://github.com/openai/gym.git
    * 把克隆的gym里的setup.py和MuJoCo相关的两行代码注释掉
    * 到克隆的gym文件夹执行shell：`pip install e .[all]`
5. VSCode直接运行jupyter可能会被拒绝，
